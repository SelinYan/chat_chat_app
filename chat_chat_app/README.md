## Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- Node.js
- npm (Node Package Manager)
- MySQL database (for backend data storage)

### Installation

1. Clone the repository:

   git clone https://github.com/SelinYan/chat_chat_app.git

2. Navigate to the project directory:

   cd chat-chat-app

3. Install dependencies:

   npm install

4. Set up the backend database by executing SQL scripts in the database folder.

## check this

mysql -u your-username -p < database/schema.sql

5. Start the backend server:

   npm run start-backend

6. Start the frontend application:

   npm start

## Usage

- Register a new account or log in if you already have one.
- Explore and connect with other users.
- Start or join conversations in the chat.
- Share posts, comment, and like.
